## 下载

### Docker

```
docker pull apachecn0/apachecn-ml-zh
docker run -tid -p <port>:80 apachecn0/apachecn-ml-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install apachecn-ml-zh
apachecn-ml-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g apachecn-ml-zh
apachecn-ml-zh <port>
# 访问 http://localhost:{port} 查看文档
```
